# standupapp
This is a demo app for showing Visual Studio "Monaco" online IDE. A WebApp for Scrum Team to host their daily stands up meeting, click on start recording and record time that each person is speaking.
## Features
| ID	| Content |
|-------|---------|
| 1     |  帮助Scrum团队完成每日立会 |
| 2     |  使用bing每日美图作为背景 |
| 3     |  背景音乐 |
